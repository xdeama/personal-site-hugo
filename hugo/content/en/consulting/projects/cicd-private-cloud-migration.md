Java and DevOps development for the private cloud migration of a major international bank. The focus was on containerizing existing workloads, automating deployment processes, and guiding migration projects through internal governance structures:

- Migrated Spring Microservices from VM-based operation to containerized environments in OpenShift, including dependent databases and observability tooling
- Automated complex deployment mechanisms for VMs using Ansible and Reactive Spring, including batch deployments with dependency ordering and automatic sidecars
- Guided the first batch of migrated systems through internal architecture and IT risk governance processes
- Analyzed, containerized, and ported an existing database anonymization solution
