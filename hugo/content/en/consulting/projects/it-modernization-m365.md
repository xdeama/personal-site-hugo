The IT infrastructure of the Habona Invest Group was historically grown, sparsely documented, and managed by an external service provider whose incentive was to extend existing dependencies rather than reduce them. I analyzed and assessed this system landscape and executed the migration to Microsoft Cloud Hybrid solutions.

The central architectural challenge was migrating production systems without any test or staging environment: the affected systems (Active Directory, Exchange Server, File Server) existed exclusively as single production instances.

### **Project Highlights**
- Introduction of Microsoft 365 including Teams and SharePoint
- User training for new working methods with Microsoft Teams and SharePoint
- Migration of on-premises systems such as Active Directory, Exchange Server, and File Server to Microsoft Cloud Hybrid solutions
